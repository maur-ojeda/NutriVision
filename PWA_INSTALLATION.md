# PWA Installation Guide - NutriVision

## Android Installation

1. **Open Chrome or Android Browser**
   - Navigate to your NutriVision app URL
   - Wait for the page to fully load

2. **Install Prompt**
   - You should see a browser prompt to "Add to Home Screen"
   - If you don't see it, tap the menu (⋮) in the top right corner
   - Select "Add to Home Screen" or "Install App"

3. **Confirmation**
   - Give the app a name if prompted
   - Confirm the installation
   - The app icon will appear on your home screen

4. **Launch**
   - Tap the NutriVision icon to launch the app
   - It will open as a standalone app, not in the browser

## iOS/iPhone/iPad Installation

1. **Open Safari**
   - Navigate to your NutriVision app URL
   - Wait for the page to fully load

2. **Share Button**
   - Tap the Share button (□ with ↑) at the bottom of the screen

3. **Add to Home Screen**
   - Scroll down and tap "Add to Home Screen"
   - The app icon and name will appear

4. **Customize (Optional)**
   - You can change the name if desired
   - Tap "Add" in the top right corner

5. **Launch**
   - The app icon will appear on your home screen
   - Tap it to launch the app as a standalone application

## macOS Installation

1. **Open Chrome or Safari**
   - Navigate to your NutriVision app URL

2. **Install in Chrome**
   - Click the install icon (↗️) in the address bar
   - Or go to Chrome menu (⋮) → "Install NutriVision"

3. **Launch**
   - The app will open as a desktop application
   - It will be available in your Applications folder

## PWA Features

Once installed, NutriVision will:

✅ **Work Offline** - Access previously loaded content without internet
✅ **Faster Loading** - Cached assets load instantly
✅ **Native Feel** - Runs as a standalone app, not in browser
✅ **Home Screen Icon** - Easy access from your device
✅ **Full Screen** - Immersive experience on mobile
✅ **Push Notifications** - (Future feature) Receive updates about your meals

## Troubleshooting

**No Install Prompt?**
- Make sure you're using a supported browser (Chrome, Safari, Edge)
- Check that your internet connection is stable
- Try refreshing the page
- On iOS, you must use Safari (not Chrome)
- Clear browser cache and try again

**App Not Working Offline?**
- First install and use the app while online
- Allow the app to cache necessary files
- Check your browser settings for offline content permissions

**Icon Not Appearing?**
- Wait a few seconds after installation
- Check your home screen/app drawer
- Search for "NutriVision" on your device

**Installation Failed?**
- Make sure you have enough storage space
- Check your internet connection
- Try a different browser
- Restart your device and try again

## Security Notes

- PWA is secure (HTTPS required)
- Your data is protected by Supabase authentication
- Offline mode doesn't expose sensitive information
- App updates automatically when you're online

## Development Notes

For developers working on the PWA:

- Service Worker is automatically generated by `vite-plugin-pwa`
- Manifest is configured in `vite.config.js`
- Icons are in the `public/` directory
- Registration happens in `src/main.tsx`
- Install prompt is handled by `PWAInstallPrompt` component

## Need Help?

If you encounter issues during installation:
1. Check browser console for errors
2. Verify you're using a supported browser
3. Make sure the app is served over HTTPS
4. Try clearing browser cache
5. Contact support with details of your device and browser
